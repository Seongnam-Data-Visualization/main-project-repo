# #3 체감이동 GAP Score

시민이 실제로 어디로 이동하는가, 그리고 그게 행정 생활권과 얼마나 다른가를 측정하는 점수.

### 사용 데이터셋

| 데이터 | 출처 | 사용 속성 예상… |
| --- | --- | --- |
| 카드 관외소비 데이터 | (신한) | 소비 발생 동 → 소비 지점 동 |
| 통계청 생활이동 OD | 통계청 | 출발 동 → 도착 동, 이동 목적 |
| 유동인구 데이터 | SKT / KT | 시간대별 체류 인구 |
| 행정동 경계 GIS | SGIS | shapefile |

### 분석 방법

**① 관외소비 비율 계산**

관외소비 비율(A동) = A동 주민의 타 동 소비액 / A동 주민 전체 소비액
과외 소비 비율이 높은 동 = 생활 자족도가 낮은 동

**실제로 그들의 소비가 어디로 빠져나가는지도 분석**

수정구 A동 주민 소비의 40%가 분당구에서 발생
→ A동의 실질 생활권은 분당구에 걸쳐 있음
→ 행정 경계(북부 생활권)와 불일치

**② 생활이동 OD 분석**

통계청 생활이동 데이터 구조
출발동 | 도착동 | 이동건수 | 이동목적(쇼핑/의료/여가 등)

→ 목적별로 분리해서 볼 것
   - 의료 목적 이동 OD : 어느 동이 의료 원정이 많은가
   - 쇼핑 목적 이동 OD : 소비 생활권 경계
   - 여가 목적 이동 OD : 문화인프라 불균형

**③ 체감 생활권 경계 역산**

각 행정동에서 이동량 Top 3 도착지를 추출
→ 연결 강도 기반으로 클러스터링 
→ 데이터가 만들어내는 "자연 생활권 경계" 도출
→ 성남시 계획 경계와 오버레이해서 불일치 면적 계산

### 체감이동 GAP Score 산출식

```
체감이동 GAP Score(행정동) =
  w₁ × 관외소비 비율
+ w₂ × 생활권 외부 이동 비율 (이동 OD 기준)
+ w₃ × 계획 생활권과 체감 생활권의 경계 불일치 여부 (0/1 또는 거리)
```
