Type Label

동별 수요-공급 불균형을 반영한 관리 유형

Type A 자족형 · 3개 동

일자리 공급, 접근성, 통근 압력이 상대적으로 균형적인 동. 생활권 내 균형 거점으로 유지·확장.

Type B 직주분리형 · 14개 동

거주 수요와 일자리 공급이 분리되어 외부 통근 압력이 큰 동. 직주 연결과 통근 시간 절감이 핵심.

Type C 접근성 불량형 · 20개 동

수요를 교통 접근성이 충분히 받치지 못하는 동. 대중교통 접근성과 환승 연결성 개선이 우선.

Type D 복합 취약형 · 13개 동

주거·노동·교통 결핍이 겹치는 동. 복합 취약지로 우선 개입 패키지가 필요.

  • Type C/D가 33개 동으로 전체의 66%를 차지
  • 동일 중생활권 내부에도 취약 원인이 다르게 나타남
  • 생활권 경계는 계획의 시작점, 동별 Type은 실행 전략의 단위

Residential

주거 분석

분당구

세대수와 아파트 비율, 매매가 중앙값이 모두 높아 주거 가격과 재고의 중심축

수정구

단독·다가구 재고와 도심형 수요가 겹치며, 자족 가능성과 취약성이 함께 나타남

중원구

연립·다세대 비중과 상대적 가격 접근성이 있으나, 접근성 보완이 주과제

Employment

노동 분석 결론

일자리 공급은 일부 거점에 집중되며, 다수 주거 동은 외부 통근에 의존합니다.

Transport

교통 분석 결론

교통은 주거-노동 불일치를 완충하며, 수요 대비 접근성이 낮은 지역이 우선 개선 대상입니다.

Planning Implication

생활권 계획에 대한 결론

2020 체계

수정중원·분당·판교 3개 대생활권은 중심지 구조를 설명하는 데 유용하지만, 동별 결핍 원인을 식별하기에는 부족합니다.

2035 개편

성남 1개 대생활권과 북부·중부·남부 3개 중생활권은 통합 생활권 관점에서 진전된 방향입니다.

보완 제안

3개 중생활권 내부에 Type A-D 관리유형을 추가해 직주분리, 접근성 부족, 복합 취약성을 별도 처방해야 합니다.
