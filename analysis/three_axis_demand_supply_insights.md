# 3-Axis Demand-Supply Insights

이 문서는 `analysis/data/seongnam_scored.csv`와 지금까지 생성한 주거·노동·교통 분석 결과를 바탕으로, 성남시를 3-axis 수요-공급 프레임으로 해석한 결론을 정리한 것이다.

## 분석 프레임

3-axis는 도시 생활권을 단순 행정구역이 아니라, 생활 수요와 도시 공급의 불균형으로 보는 접근이다.

| Axis | 수요 | 공급 | 주요 해석 |
|---|---|---|---|
| 주거 | 세대수, 총인구, 생산가능인구 | 주택 재고, 주거비, 주택 유형 | 어디에 주거 수요가 집중되고, 어떤 주거 공급 구조가 이를 받치는가 |
| 노동 | 생산가능인구, 거주 기반 노동 수요 | 지역 내 종사자, 사업체, 법인 | 거주 인구가 지역 내 일자리와 얼마나 연결되는가 |
| 교통 | 통근량, 서울/축 방향 이동 수요 | 대중교통 접근성, 통행 수단 구조 | 주거-노동 불일치를 교통 공급이 얼마나 완충하는가 |

통합 분석에서는 `total_workers / 생산가능인구_19_64세`를 일자리 공급 격차의 핵심 지표로 보고, `axis_commute_share + seoul_commute_share`를 외부 통근 압력으로 보았다. `accessibility_score`는 교통 공급 측면의 완충 능력으로 해석했다.

## 생활권 계획 맥락

기존 2020 도시기본계획은 성남을 다음 3개 대생활권으로 구분했다.

- 수정중원 대생활권
- 분당 대생활권
- 판교 대생활권

2035 도시기본계획은 생활권을 다음 구조로 개편한다.

- 성남 1개 대생활권
- 북부 / 중부 / 남부 3개 중생활권

3-axis 분석 결과는 2035년식 개편 방향, 즉 성남을 하나의 대생활권으로 보고 내부를 기능적 중생활권으로 다시 나누는 접근이 타당하다는 점을 뒷받침한다. 주거, 노동, 교통의 불균형이 기존 행정구 또는 기존 3중심 체계 안에서만 닫히지 않고, 동 단위로 교차하기 때문이다.

## Type Label 요약

`seongnam_scored.csv`의 `type_label`은 동별 수요-공급 불균형을 네 유형으로 구분한다.

| Type | 의미 | 해석 |
|---|---|---|
| Type A (자족형) | 가장 양호 | 일자리 공급, 접근성, 통근 압력이 상대적으로 균형적 |
| Type B (직주분리형) | 위험 증가 | 거주 수요와 일자리 공급이 분리되고 외부 통근 압력이 큼 |
| Type C (접근성 불량형) | 위험 높음 | 수요를 교통 접근성이 충분히 받치지 못함 |
| Type D (복합 취약형) | 위험 가장 높음 | 복수 축의 불균형이 겹치는 취약 유형 |

전체 50개 행정동의 분포는 다음과 같다.

| Type | 동 수 |
|---|---:|
| Type A (자족형) | 3 |
| Type B (직주분리형) | 14 |
| Type C (접근성 불량형) | 20 |
| Type D (복합 취약형) | 13 |

구별 분포는 다음과 같다.

| 구 | Type A | Type B | Type C | Type D | 해석 |
|---|---:|---:|---:|---:|---|
| 분당구 | 0 | 10 | 6 | 6 | 주거비와 통근 압력이 큰 직주분리형 생활권 |
| 수정구 | 3 | 3 | 7 | 4 | 일부 자족 거점과 접근성 취약지가 공존 |
| 중원구 | 0 | 1 | 7 | 3 | 접근성 불량형 비중이 큰 보완 필요 생활권 |

## 핵심 결론

### 1. 성남은 하나의 대생활권으로 묶되, 내부 처방은 세분화해야 한다

2035 개편의 `성남 대생활권` 방향은 데이터상 설득력이 있다. 주거 수요, 일자리 공급, 통근 흐름이 구 경계를 넘어 연결되기 때문이다. 다만 하나의 생활권으로 묶는 것만으로는 충분하지 않다. 동별 `type_label`을 보면 C와 D가 33개 동으로 전체의 66%를 차지한다. 즉 도시 전체를 하나의 생활권으로 통합하되, 내부에서는 취약 유형별로 다른 공급 전략이 필요하다.

### 2. 분당은 주거 가격과 통근 압력이 결합된 직주분리형 생활권이다

분당구는 세대수와 주거 가격 측면에서 성남의 핵심 주거축이다. 그러나 Type A가 없고 Type B가 10개 동으로 가장 많다. 이는 분당이 단순히 주거 여건이 좋은 지역이라기보다, 고가 주거지와 외부 통근 압력이 결합된 생활권임을 의미한다.

따라서 분당권의 핵심 과제는 주택 공급량 자체보다 `직주근접성`, `통근 수요 분산`, `주요 업무지 접근성`이다. 특히 기존 2020 체계에서 분당과 판교를 별도 생활권으로 보는 관점은 여전히 의미가 있지만, 데이터상으로는 분당 내부에서도 직주분리형·복합취약형 동이 섞여 있으므로 더 미세한 중생활권 관리가 필요하다.

### 3. 수정구는 자족 가능성과 취약성이 동시에 나타나는 전환 생활권이다

수정구는 Type A가 확인되는 유일한 구이지만, 동시에 Type C와 D도 많다. 이는 수정구가 완전히 취약한 생활권이라기보다, 자족 가능성이 있는 일부 동과 접근성·복합 취약 동이 강하게 갈리는 구조라는 뜻이다.

따라서 수정구는 북부 또는 원도심 생활권 재편에서 `일자리-주거 연계 강화`와 `취약지 접근성 개선`을 동시에 다뤄야 한다. 자족형 동은 생활권 내 거점으로 활용하고, C/D 동은 교통 공급과 생활 SOC 접근성을 보완하는 방식이 적합하다.

### 4. 중원구는 접근성 보완이 가장 직접적인 정책 과제다

중원구는 Type C가 7개, Type D가 3개로 접근성 또는 복합 취약 유형이 대부분이다. 일자리 공급이 특정 산업지에 존재하더라도, 생활권 전체 관점에서는 교통 접근성이 수요를 충분히 받치지 못하는 동이 많다.

따라서 중원구는 단순 주택 공급보다 `교통 접근성 개선`, `산업지-주거지 연결`, `버스·철도 접근 보완`이 우선 과제다. 2035년의 중생활권 체계에서는 중원구를 수정구와만 묶어 보는 기존 수정중원 관성보다, 북부/중부/남부 연결축 안에서 어느 축의 접근성이 약한지 보는 방식이 더 유용하다.

### 5. 판교는 독립 중심지이지만 주변 주거지와의 수요-공급 불균형을 함께 봐야 한다

기존 2020 계획에서 판교는 별도 대생활권으로 설정될 만큼 강한 중심성을 가진다. 그러나 3-axis 프레임에서는 판교를 일자리 중심지로만 보면 부족하다. 판교 주변 주거 동은 높은 가격, 통근 압력, 접근성 차이가 동시에 나타날 수 있다.

따라서 판교권은 `업무 중심성`뿐 아니라 `주거비 부담`, `주변 동의 통근 의존`, `대중교통 접근성`을 함께 보는 관리 단위가 되어야 한다. 판교 중심 기능이 성남 전체로 파급되려면, 주변 주거 생활권과 연결되는 교통 공급이 같이 강화되어야 한다.

### 6. 생활권 개편의 핵심은 중심지 개수보다 축별 결핍을 찾는 것이다

2020년의 3중심 체계는 도시 구조를 이해하는 데 유용하지만, 실제 정책 처방은 “어느 중심에 속하는가”보다 “어떤 축의 공급이 부족한가”가 더 중요하다.

- Type B는 일자리와 주거의 분리 문제다.
- Type C는 교통 접근성 공급 문제다.
- Type D는 복수 결핍이 겹친 문제다.
- Type A는 향후 생활권 거점 또는 균형 모델로 삼을 수 있다.

따라서 2035 생활권 개편은 `성남 대생활권`이라는 통합 틀 아래에서, 북부/중부/남부 중생활권 각각에 대해 Type B/C/D의 원인이 무엇인지 분리 진단하는 방식으로 운영되어야 한다.

## 정책적 시사점

### 주거 Axis

- 분당은 주거비와 수요 집중이 강하므로 추가 공급 논의와 함께 통근 부담을 줄이는 정책이 필요하다.
- 수정구와 중원구는 단순 저가 주거지가 아니라 접근성 보완 여부에 따라 생활권 경쟁력이 달라지는 지역이다.
- 동별 주거 수요는 행정구 평균보다 훨씬 이질적이므로, 구 단위 평균만으로 생활권을 설계하면 취약 동을 놓칠 수 있다.

### 노동 Axis

- 일자리 공급은 판교·분당 및 일부 산업지에 집중되어 있고, 많은 주거 동은 외부 일자리 의존도가 높다.
- `total_workers / 생산가능인구`가 낮은 동은 주거 수요는 있으나 지역 내 노동 공급이 부족한 동으로 해석할 수 있다.
- Type B 동은 신규 주택 공급보다 직주 연결, 업무 접근성, 통근 시간 절감이 우선이다.

### 교통 Axis

- 교통은 주거와 노동의 불일치를 보완하는 공급 장치다.
- Type C와 D가 많은 것은 현재 접근성 공급이 수요를 충분히 완충하지 못하는 동이 많다는 뜻이다.
- 교통 투자 우선순위는 단순 통행량보다 `높은 수요 + 낮은 접근성 + 높은 외부 통근`이 겹치는 동을 중심으로 잡는 것이 합리적이다.

## 최종 결론

성남시 생활권은 더 이상 `수정중원-분당-판교`라는 중심지 구분만으로 설명하기 어렵다. 현재 데이터는 성남이 하나의 통합 생활권으로 작동하면서도, 내부적으로는 주거-노동-교통 공급 격차가 동 단위로 강하게 분화되어 있음을 보여준다.

따라서 2035년 생활권 개편의 실질적 과제는 `1개 대생활권 + 3개 중생활권`이라는 공간 구조를 선언하는 데서 끝나지 않는다. 각 중생활권 안에서 Type B, C, D를 식별하고, 직주분리·접근성 부족·복합 취약성을 각각 다른 정책 패키지로 대응하는 것이 핵심이다.

한 줄로 정리하면, **성남의 생활권 전략은 중심지 배치에서 수요-공급 격차 관리로 이동해야 한다.**
