# #1 수요-공급 불일치 score

아래 세가지 기준을 모두 만족하는 수요-공급 대상을 선정하였다. 

- 수요 집단이 **특정 시설에 의존도가 높은** 그룹인가
- 수요와 공급 데이터를 **같은 공간 단위(행정동)** 로 맞출 수 있는가
- 신도시 vs 구시가지 **격차가 실제로 존재할 가능성**이 있는가

아래 제시한 4가지 부문에 대해 동일한 분석 틀(수요 지표 → 공급 지표 → 비율 계산 → choropleth 지도)로 만들면, 나중에 합쳤을 때 **비교 가능한 구조가 될 수 있을 것 같다.** 

### 1. 고령인구 x 노인 복지 시설

|  | 내용 |
| --- | --- |
| 수요 지표 | 행정동별 65세 이상 인구 비율 |
| 공급 지표 | 노인요양시설, 경로당, 노인복지관 수 |
| 핵심 지표 | 고령인구 1,000명당 노인시설 수 |
| 기대 결과
(데이터 분석을 통해 실제로 확인해 봐야함) | 수정·중원구 고령화율이 높지만 시설 밀도는 분당보다 낮을 가능성  |

### 2. 장애인 등록 인구 × 장애인복지시설

|  | 내용 |
| --- | --- |
| 수요 지표 | 행정동별 장애인 등록 인구 수 |
| 공급 지표 | 장애인복지관, 주간보호센터, 공동생활가정 수 |
| 핵심 지표 | 장애인 인구 1,000명당 장애인복지시설 수 |
| 기대 결과 | 장애인 인구는 분산돼 있는데 시설은 특정 권역에 집중 |

### 3. 아동인구 × 돌봄·교육시설

|  | 내용 |
| --- | --- |
| 수요 지표 | 행정동별 0~12세 아동 인구 수 |
| 공급 지표 | 어린이집 정원 수, 지역아동센터 수 |
| 핵심 지표 | 아동 100명당 어린이집 정원 수 |
| 기대 결과 | 위례·판교 신도시는 아동 많고 시설도 많음 vs 수정구는 아동 줄어도 시설 공급 따라가지 못하는 구조 |

### 4. 기초생활수급자 × 의료·복지 접근성

|  | 내용 |
| --- | --- |
| 수요 지표 | 행정동별 기초생활수급자 수 + 1인 가구 비율 |
| 공급 지표 | 의원·보건소 수, 종합사회복지관 수 |
| 핵심 지표 | 수급자 1,000명당 의료·복지 공급 수 |
| 기대 결과 | 수급자 밀집 지역(구시가지)에 오히려 공급이 부족한 역설 |
